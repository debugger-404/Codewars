git config --global core.autocrlf false отключает автоматическое преобразование символов новой строки (CRLF)
при работе с Git. При установке значения "false", Git не будет автоматически изменять символы новой строки при коммитах
и проверках кода. Это может быть полезно, если вы работаете на разных операционных системах, где используются различные
символы новой строки.
